# Projeto Currículo Online - Capstone 1

Este é o meu primeiro projeto oficial do **The Complete Web Development Bootcamp** da Angela Yu. O objetivo foi aplicar os conceitos fundamentais de HTML5 e CSS3 para criar um currículo online profissional e responsivo.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e organização de conteúdo.
* **CSS3:** Estilização moderna, efeitos de hover e layout responsivo.
* **Font Awesome:** Ícones profissionais para redes sociais.
* **GitHub Pages:** Hospedagem e deploy contínuo.

## Funcionalidades

* **Design Minimalista:** Focado na leitura e clareza das informações.
* **Interatividade:** Efeitos de transição suave (hover) nos links e ícones.
* **Multi-página:** Navegação entre a página principal, hobbies e contato.
* **Favicon Personalizado:** Identidade visual na aba do navegador.
